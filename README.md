# Purpose

Native bindings to clue to help wrap libraries in a C interface.

# Build dependencies

- CMake
- C++14 compiler
- Qt 5.9 or newer

# Build

```sh
git clone https://github.com/stephaneyfx/clue-sys.git
cd clue-sys
cargo build
```

# Platform support

All, to the extent of the availability of the dependencies.
